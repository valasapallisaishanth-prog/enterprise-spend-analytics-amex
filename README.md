# Enterprise Spend Analytics & Reporting (AmEx-Style)

## Business Objective
Analyze enterprise transaction-level spend data to track performance KPIs, identify spend drivers across suppliers, categories, and regions, and support data-driven decision-making.

## Tools & Technologies
- SQL (SQLite)
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib)

## Key KPIs
- Total enterprise spend
- Month-over-month spend growth
- Spend by category and region
- Supplier spend concentration

## Key Insights
- Identified high-spend categories and regions driving enterprise costs
- Observed consistent month-over-month spend trends
- Detected supplier concentration risk, highlighting optimization opportunities

## Business Value
This analysis enables leadership teams to monitor spend performance, manage supplier risk, and identify cost optimization opportunities through data-driven insights.
